# LightEngine
Petit moteur basique issu d'un apprentissage d'openGL

1ere phase : développement d'un visualiseur 3d prenant des objets au format .obj

2eme phase : développement d'un moteur physique de base

3eme phase : développement de la prise en charge des contrôles (1, clavier; 2, joystick; 3, tactile)
